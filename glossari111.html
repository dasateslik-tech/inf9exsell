Адрес ячейки
Состоит из названия столбца и номера строки, на пересечении которых находится ячейка. Например, Е23 или МL123.

Абсолютная ссылка
Фиксированный адрес ячейки, который не меняется при растягивании формулы. Обозначается знаком $. Если необходимо, чтобы при копировании ссылки не изменялись, то нужно использовать абсолютную ссылку.
🔗 Урок по ссылкам в Excel
Автосумма
Это инструмент в Excel, позволяющий автоматически суммировать диапазон чисел (в строке или столбце) без ручного ввода формул. Она автоматически определяет ячейки с числами, создает формулу =СУММ() и выводит результат.

Диапазон
Область из нескольких выделенных ячеек. Адрес диапазона определяется адресом верхней левой и нижней правой ячеек выделенной области, например: B2:D5.
Диспетчер имен
Инструмент для создания «понятных» имен для ячеек вместо их адресов.

Относительная адресация
В электронных таблицах применяется принцип относительной адресации: при копировании формулы на новое место ссылки на ячейки изменяются относительно перемещения формулы.
Относительная ссылка
Ссылки в формулах, которые изменяются при копировании.

Условное форматирование
Автоматическое изменение формата ячеек в зависимости от их содержимого.Например, в предложенном диапазоне чисел выделить красным все отрицательные числа. Для этого: выделите данные, выполните «Главная»-«Условное форматирование»-«Правила выделения ячеек». Выберите критерии: «Меньше», в поле введите 0.
Фильтр
Средство отображения только тех строк таблицы, которые соответствуют заданным условиям.
Формула
Запись выражения, по которому выполняются вычисления. Математическое или логическое выражение, которое всегда начинается со знака «=».
